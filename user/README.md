# User

Responsible for user registration, login, and other functions.

## Run In Local

1. Initialize .env File

The following configuration is required:

```
APP_NAME=Goravel
APP_ENV=local
APP_KEY=
APP_DEBUG=true

GRPC_HOST=
GRPC_PORT=

JWT_SECRET=

DB_CONNECTION=postgresql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=goravel
DB_USERNAME=goravel
DB_PASSWORD=123123
```
