Authentication

Add the `Authorization` header the request, the authorization token is generated by EmailLogin and EmailRegister endpoints. The token should be stored locally and added to the header of all subsequent requests, it will be returned by the header of all subsequent requests that need to be verified. It may be fresh in the process, so it should be updated locally when the requestâ€™s header contains the Authorization key.
